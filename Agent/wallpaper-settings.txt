LOCATION WALLPAPER SETTINGS
===========================

OFF     No wallpaper
STILL   Single static capture, no animation
BASIC   4 pan images (N/E/S/W), CSS animation
ORBIT   Live Mapbox rendering, camera orbits


BASIC MODE
----------

Camera:
  Venue/Address  zoom 16, pitch 70°
  City           zoom 11, pitch 70°
  Bearings       0°, 90°, 180°, 270° (N/E/S/W)

Capture size:
  Width          container + 100px
  Height         container + 300px

Animation:
  Slide          20 seconds
  Pan            100px leftward over 20s (left edge starts at container left)
  Crossfade      1.5 seconds

Loading:
  All 4 images captured in parallel at start
  Animation begins when all 4 are ready

Container growth:
  300px buffer handles growth without stretch
  If exceeded: image stretches temporarily
  Recapture runs while another image is visible
  Fresh image ready next cycle


CSS (components-new.css):
  .component-locationwallpaper-basic-container
  .component-locationwallpaper-basic-image
  .component-locationwallpaper-basic-image--active
  @keyframes locationwallpaper-basic-pan (20s, 0 to -100px)

JS (components-new.js):
  startBasicMode(lat, lng)
  advanceBasic()
  captureMap(camera, w, h, cb)
  stopBasicMode()
  removeBasicMode()
