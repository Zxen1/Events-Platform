### Known Edge Case ŌĆö Backup Restores
Restoring a database backup will resurface soft-deleted accounts and posts. The cron will re-process them on its next run if they are past the grace period. CDN images deleted before the restore are permanently gone ŌĆö restored DB records will reference missing files. **Rule:** database backups must never be older than the grace period (7 or 30 days, whichever applies). Restoring an older backup may require manual cleanup. This is an accepted operational risk ŌĆö no code solution exists for it.

Any future registration attempt with the same email will see the tombstone and allow a fresh account.

### How `send-email.php` Works
- Called via the gateway like all other connectors
- POST params: `message_key`, `to_email`, `to_name`, `member_id`, plus any placeholder values declared in the template

- Injects logo header dynamically into email body
- Sends via PHPMailer (SSL, port 465)
- Logs result to `emails_sent` table regardless of success or failure
